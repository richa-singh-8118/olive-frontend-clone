# Olive 🫒

A pixel-perfect, mobile-responsive landing page for **Olive**, a nutrition-tracking and ingredient-scanning application. This project showcases modern web design principles with smooth animations, a clean UI, and high-performance React implementation.

🚀 **Live Demo:** [https://olive-frontend-clone-eight.vercel.app](https://olive-frontend-clone-eight.vercel.app)

---

## ✨ Features

- **📱 Pixel-Perfect Design**: Meticulously crafted UI matching professional design standards.
- **✨ Smooth Animations**: Powered by **Framer Motion** for an engaging and fluid user experience.
- **🔍 Scan & Identify**: Interactive highlights showing how the app scans ingredients and provides health scores.
- **📊 Health Benefits**: Visual representation of nutritional insights and personalized scoring.
- **🌗 Responsive Layout**: Optimized for all screen sizes, from mobile devices to desktop monitors.
- **⚡ Performance First**: Built with **Vite** for ultra-fast development and optimized production builds.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/richa-singh-8118/olive-frontend-clone.git
   cd olive-frontend-clone
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` to see the app in action.

## 📦 Build for Production

To create an optimized production build:

```bash
npm run build
```

The output will be in the `dist` folder, ready to be deployed.
